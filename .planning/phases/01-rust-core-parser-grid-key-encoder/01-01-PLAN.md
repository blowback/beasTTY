Module responsibilities (locked by D-19; implementations come in Plans 04/05):
- `terminal.rs`: VT52 semantic layer. Owns Grid + Scrollback + Dirty + cursor + bell_pending + host_reply.

- `scrollback.rs`: VecDeque<Row> ring with 10k-line default cap (D-11, D-12).
- `dirty.rs`: Byte-per-row dirty bitmap for zero-copy JS alignment in Phase 2.
- `vt52.rs`: Parser state machine; dispatch to Terminal methods. Winner of Plan 03 spike fills this.
- `key.rs`: Stateless `encode(KeyEvent) -> Vec<u8>` (D-13).
