|----------|-------------|
| feature-detect → body mutation | The `typeof navigator.serial === 'undefined'` check gates a destructive `document.body.innerHTML = ...` — a false-positive would wipe the app |
| polite-fail HTML → user browser | renderPoliteFail's innerHTML string is static; but if any dynamic insertion (URL fragment, search param) is ever added, it becomes an XSS surface |
| module scope → global title/body | renderPoliteFail mutates `document.title` and `document.body` — a single-responsibility function; no other Phase 5 code should mutate these outside designated paths |
